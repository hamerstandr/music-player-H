# 🎵 Music Player H - موزیک پلیر حرفه‌ای

## معرفی
Music Player H یک پخش‌کننده موسیقی و ویدیو حرفه‌ای و مدرن برای ویندوز است که با فناوری WPF و C# و .NET 10 توسعه یافته است. این برنامه با طراحی کاربرپسند و امکانات پیشرفته، تجربه‌ای بی‌نظیر از پخش رسانه را ارائه می‌دهد.

## ✨ ویژگی‌های اصلی

### پخش رسانه
- ✅ پشتیبانی از فرمت‌های صوتی: MP3, WAV, FLAC, AAC, OGG, WMA
- ✅ پشتیبانی از فرمت‌های تصویری: MP4, AVI, MKV, MOV, WMV, FLV, WebM
- ✅ کنترل‌های کامل پخش (Play, Pause, Stop, Next, Previous)
- ✅ نوار پیشرفت قابل جستجو
- ✅ کنترل حجم صدا و Mute
- ✅ پخش خودکار ترک بعدی

### لیست پخش
- ✅ افزودن فایل‌های تکی یا چندگانه
- ✅ افزودن کل پوشه به لیست پخش
- ✅ حذف ترک‌های انتخابی
- ✅ پاک کردن کل لیست پخش
- ✅ نمایش اطلاعات ترک (عنوان، هنرمند، آلبوم، مدت زمان)

### اتصال به TrafficWatch
- ✅ اتصال خودکار به سرویس TrafficWatch از طریق Named Pipe
- ✅ **عدم وابستگی**: برنامه بدون TrafficWatch هم به طور کامل کار می‌کند
- ✅ ارسال بلادرنگ اطلاعات آهنگ در حال پخش
- ✅ Heartbeat خودکار هر 30 ثانیه
- ✅ تلاش مجدد برای اتصال در صورت قطع شدن
- ✅ نمایش وضعیت اتصال در UI

### رابط کاربری چندزبانه
- ✅ **فارسی** (زبان پیش‌فرض)
- ✅ **عربی**
- ✅ **English**
- ✅ تغییر زبان به صورت بلادرنگ بدون نیاز به راه‌اندازی مجدد

### رابط کاربری
- ✅ طراحی مدرن با تم تاریک (Dark Theme)
- ✅ انیمیشن‌های نرم و واکنش‌گرا
- ✅ نمایش ویدیو با کیفیت بالا
- ✅ نشانگر وضعیت اتصال به TrafficWatch

## 🔧 معماری فنی

### تکنولوژی‌ها
- **.NET 10** - آخرین نسخه فریم‌ورک دات‌نت
- **WPF** - برای رابط کاربری دسکتاپ
- **LibVLCSharp** - موتور پخش رسانه قدرتمند
- **CommunityToolkit.Mvvm** - پیاده‌سازی الگوی MVVM
- **Named Pipes** - ارتباط با TrafficWatch

### ساختار پروژه
```
MusicPlayerH/
├── App.xaml                    # تنظیمات برنامه و استایل‌ها
├── App.xaml.cs                 # کد پشت صحنه برنامه
├── MainWindow.xaml             # رابط کاربری اصلی
├── MainWindow.xaml.cs          # کد پنجره اصلی
├── MusicPlayerH.csproj         # فایل پروژه
└── src/
    ├── MainViewModel.cs        # ViewModel اصلی با منطق کسب‌وکار
    ├── TrafficWatchPluginClient.cs  # کلاینت ارتباط با TrafficWatch
    ├── Models/
    │   └── MediaTrack.cs       # مدل ترک رسانه
    ├── Converters/
    │   └── BoolConverters.cs   # مبدل‌های XAML
    └── Resources/
        └── Languages/          # فایل‌های ترجمه
            ├── fa.xaml         # فارسی
            ├── ar.xaml         # عربی
            └── en.xaml         # انگلیسی
```

## 🚀 نحوه اجرا

### پیش‌نیازها
- ویندوز 10/11
- .NET 10 SDK
- Visual Studio 2026 (اختیاری)

### اجرای مستقیم
```bash
cd MusicPlayerH
dotnet build
dotnet run
```

### اجرا در Visual Studio
1. پروژه را در Visual Studio 2026 باز کنید
2. مطمئن شوید Target Framework روی net10.0-windows تنظیم شده
3. کلید F5 را بزنید

### اتصال به TrafficWatch (اختیاری)
اگر سرویس TrafficWatch در حال اجرا باشد، برنامه به طور خودکار متصل می‌شود:

```bash
# شروع سرویس TrafficWatch (اگر نصب است)
sc start TrafficWatch
```

برنامه حتی اگر TrafficWatch اجرا نباشد، به طور کامل کار می‌کند.

## 🎯 قابلیت‌های اضافی

- **پخش ویدیو**: نمایش ویدیو با کیفیت بالا در پنجره اصلی
- **Drag & Drop**: (قابل گسترش) کشیدن و رها کردن فایل‌ها
- **کلیدهای میانبر**: (قابل گسترش) کنترل با کیبورد
- **Metadata Reading**: (قابل گسترش) خواندن اطلاعات ID3 از فایل‌های موسیقی
- **Equalizer**: (قابل گسترش) اکولایزر حرفه‌ای
- **Themes**: (قابل گسترش) تم‌های مختلف رنگی
- **Multi-language**: پشتیبانی از سه زبان فارسی، عربی و انگلیسی
