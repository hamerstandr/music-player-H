# Music Player H - موزیک پلیر حرفه‌ای

## معرفی
Music Player H یک پخش‌کننده موسیقی حرفه‌ای و مدرن برای ویندوز است که با فناوری WPF و C# توسعه یافته است. این برنامه با طراحی کاربرپسند و امکانات پیشرفته، تجربه‌ای بی‌نظیر از پخش موسیقی را ارائه می‌دهد.

## ویژگی‌ها

### 🎵 پخش موسیقی
- پشتیبانی از فرمت‌های مختلف صوتی
- کنترل کامل پخش (Play, Pause, Next, Previous)
- نمایش اطلاعات آهنگ در حال پخش
- نوار پیشرفت هوشمند

### 📋 لیست پخش
- مدیریت لیست پخش دلخواه
- افزودن آهنگ‌ها به لیست
- ناوبری آسان بین ترک‌ها
- نمایش اطلاعات کامل هر ترک

### 🔗 اتصال به TrafficWatch
- ارتباط بلادرنگ با سرویس TrafficWatch از طریق Named Pipe
- ارسال خودکار اطلاعات آهنگ در حال پخش
- نمایش وضعیت اتصال در رابط کاربری
- Heartbeat خودکار هر 30 ثانیه
- ثبت نام خودکار افزونه

## معماری فنی

### تکنولوژی‌ها
- **.NET 10** - آخرین نسخه فریم‌ورک دات‌نت
- **WPF** - برای رابط کاربری مدرن
- **C#** - زبان برنامه‌نویسی
- **Named Pipes** - برای ارتباط با TrafficWatch

## نحوه کار با TrafficWatch

### پروتکل ارتباطی

برنامه از طریق Named Pipe با نام `TrafficWatchPluginPipe` به سرویس TrafficWatch متصل می‌شود.

#### پیام‌های ارسالی:

1. **ثبت نام (Register)**
```json
{
  "action": "register",
  "name": "Music Player H",
  "version": "2.0.0",
  "icon": "🎵"
}
```

2. **داده استریم (Stream Data)**
```json
{
  "action": "stream_data",
  "timestamp": "2024-01-15T10:30:00.000Z",
  "payload": {
    "type": "now_playing",
    "title": "Bohemian Rhapsody",
    "artist": "Queen",
    "album": "A Night at the Opera",
    "duration": "5:55",
    "progress": 45,
    "isPlaying": true
  }
}
```

3. **Heartbeat**
```json
{
  "action": "heartbeat"
}
```

### راه‌اندازی

1. ابتدا سرویس TrafficWatch را اجرا کنید:
```bash
sc start TrafficWatch
```

2. سپس برنامه Music Player H را اجرا کنید.

3. برنامه به صورت خودکار به TrafficWatch متصل شده و ثبت نام می‌کند.

## رابط کاربری

### بخش‌های اصلی:

1. **هدر برنامه**
   - عنوان برنامه
   - وضعیت اتصال به TrafficWatch

2. **لیست پخش** (سمت چپ)
   - نمایش تمام آهنگ‌ها
   - انتخاب آهنگ برای پخش

3. **اطلاعات آهنگ در حال پخش** (سمت راست)
   - کاور آلبوم
   - عنوان، هنرمند و آلبوم
   - نوار پیشرفت

4. **کنترل‌های پخش** (پایین)
   - دکمه قبلی
   - دکمه Play/Pause
   - دکمه بعدی

## نکات مهم

1. **امنیت**: Named Pipe فقط به کاربران محلی اجازه اتصال می‌دهد
2. **Performance**: داده‌های استریم سبک هستند (JSON کوچک)
3. **Heartbeat**: هر 30 ثانیه به صورت خودکار ارسال می‌شود
4. **قطع اتصال**: در صورت قطع اتصال، برنامه سعی می‌کند مجدداً متصل شود

## عیب‌یابی

| مشکل | راه حل |
|------|--------|
| اتصال برقرار نمی‌شود | مطمئن شوید سرویس TrafficWatch در حال اجرا است |
| وضعیت اتصال قرمز است | بررسی کنید سرویس TrafficWatch فعال باشد |
| داده‌ها ارسال نمی‌شوند | لاگ کنسول را بررسی کنید |
